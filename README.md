# 🔐 React + Supabase Authentication

## 📸 Screenshot

![Sign in](./public/Signin.PNG)

A simple and secure authentication system built with **React** and **Supabase**, featuring:

✅ User Sign Up  
✅ User Sign In  
✅ User Sign Out  
✅ Protected Routes  
✅ Auth Context (global state)

---

## 🚀 Tech Stack

- **React 19**
- **React Router v7**
- **Supabase Auth**
- **Context API**
- **Tailwind CSS**
